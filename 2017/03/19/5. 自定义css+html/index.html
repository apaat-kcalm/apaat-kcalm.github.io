#5 自定义css+html

發表於 2017-03-19

目标

学习目标

成果

html和css的相关基础概念整理

行动

安排进程

3.5 html
刷简单的在线教程（html+css）
https://www.codecademy.com/zh/

3.6 css
刷慕课笔记+在线教程（幕布）http://www.imooc.com/article/5253

3.7 概念疑惑的地方做一遍

3.8 用css~+html，美化一篇copy大大的文章,遇到困难不会，查！
参考大大的排版流程 https://shimo.im/doc/j1s3Pe3o2pQOPVqg

工具

慕课网、3wschool在线编辑器

经验总结：

初步认识html+css

html是标记语言,是个骨架
head和body
标签作用和makedown的语法有点相似

css，是装饰（颜色，大小，行距，字间距…）html的。对着骨架填
选择器，权重，css写入html方式，标签属性（id,class,type…），
排版（缩进什么的，大小什么的），盒模型，
元素

记录

用html标记文章

注意嵌套时标签的开闭（类似括号，成双成对）

  • p—段落标签
  • span—单独做不同样式的标签

有序列表

  • ul—li—/li—ul
    无序列表
  • ol—li—/li—/ul>

表格

  • table—tr—td—td—/td—/td—/tr—table

标题1

  • h1

加粗

  • strong

用css美化

善用选择器

  • 权重100 id选择器
  • 权重10 类选择器
  • 权重一样，近标签者胜

盒模型（上右下左）

  • border 边框
  • padding 填充
  • margin 段距
  • 话在哪里，标签属性就加那~（靠近内容所在标签）

css写入html
嵌入式(html文件，head处)

  • head—style type=”text/css”>样式—（闭合style）
    内联式
  • p style=”样式”>内容–(闭合p)

问题

margin左右缩进与span标签里头加strong标签出现空格—
不缩进正常

实际情况

反思前提假设
T假设

  • 教程能够指导自己学会html,css的使用。打算刷w3school的教程
    修正：
  • 教程也有优劣，适用范围，不同的用途，不能一概而论。w3school概念太零散，不够系统，只能当字典~
    — 刷慕课网关于html和css

假设

  • 凭借自身有限的了解，以为how能够支撑what

实际做的过程中，比想象的复杂很多。需要掌握的未知知识比想象中的多，又很杂
修正：

  • 做计划前好好反思假设！问自己 还有没有？
    — 刷慕课网关于html和css的笔记，整理过一遍作为知识储备
  • 用幕布做一笔记，针对不太会的实践的过一遍

受挫，情绪干扰~
（允许自己发泄一阵，发泄完后想怎么解决！重新就原假设原计划结合反馈反思）

一阵好一阵无力的精力，容易崩盘。

  • （节奏！）

复盘

思路很重要，每一步的规划也很重要。具体的操作复杂度出乎意料~
——注意细节！不要一概而论~不要想当然~
设立目标不到位，还需使劲练习~（模型，假设）
设立截止时间。目标细化到每一个小的目标，以目标的可完成时间制定日计划！

#3 Markdown Here 实现高效排版

目标

  • 掌握排版需要的css最基本参数的知识
  • 字体大小、行间距、字间距、字体颜色)
  • 掌握整个流程，抓重点

参考

https://github.com/veganshe/CodeBlock/blob/master/Markdown-here/markdown-here-lxl.css

http://www.veganshe.com/tech/use-markdown-here-to-layout/

makedown here

  • 分隔内容（makedown书写）和样式（颜色，大小，距离…）
  • 一键渲染功能 更加快捷省心，不用一个地方一个地方的调（注释、重点的颜色，行间距什么的）。
  • 只要专注于内容就ok

标准1

  • 安装makedown-here插件
  • 右键设置 [基本渲染css],粘贴css样式
  • 文章粘贴—微信公众号编辑器
    一键渲染（ctrl+shift+M）

diy记录

why

标配在手机端丑——标题大小偏大，加粗颜色单一

改哪里

字体大小、行间距、字间距、字体颜色

行动

找个教程，粗看一遍，不懂问号，不懂哪里就找哪里。再做一遍
提示

自定义 CSS 里的设置有时不起作用试试在后面补上!important，

h2 { font-size: 20px !important; }

代码高亮省略

问题
把要设置的参数都摆上去（删除其他），发现行距，段距不行，太挤了~

没有设置外边距（上右下左）
margin:20px 0 10px;(段距~)

把各级标题统一设置了段距，1级标题无用——单独设置段距值，强制执行~

border-bottom:1px solid #ddd
makedown相当于*(固体填充#ddd，1px粗的线)

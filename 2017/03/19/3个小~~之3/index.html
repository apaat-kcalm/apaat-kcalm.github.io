#4 3个小~~之创客贴

目标

学习目标

目的

快速作图

过程（把创客贴的自带教程过了一遍）

基本操作使用

技巧

文字

要点：

  • 文字（颜色，字体）与图片（主题，颜色；结构）协调，有辨识度（深背景-浅字体）
  • 对齐,旋转，透明度（阴影、蒙版）
  • 裁切感

色彩

  • 黑灰百搭
  • 临近色和谐（一排）
  • 同类色统一——H一样，S不同

色彩3种以内

  • 可以参考图片的主题
  • 背景繁杂，突出重点内容（不一样的颜色、更醒目的颜色（对比色，深））
  • 多张图片
  • 整体环境颜色

图片

  • 工具
  • 图片吸附框（一张）
  • 组件（多张）

滤镜
专业一点可以看画面的颜色（RGB三原色的多少），选择滤镜
亮度：黑白，褪色（干净大气），胶片（更细腻），午茶
R:怀旧，玫瑰(色彩浓郁)
G:校园，森林
B:日系，蓝调
盛夏（暖）

感想：

审美，创意这东西一时急不来~

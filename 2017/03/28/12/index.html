搜索技巧

网页构成

标题
url
摘要

目的

更省力、快的找到答案

思路

确定搜索什么

在什么样的网站以什么样的形式存在，会出现哪些特别的关键词

构建搜索关键词

结合网页特征
表述准确
简练
根据反馈不断优化调整
顺藤摸瓜

搜索

根据反馈评估，优化关键词

get

搜索引擎

技巧

百度特有

搜索论坛版块
百度
『 』
精准匹配《》
电影、书籍、音乐
根据目标特定，换语言搜索

标题

intitle:
网页标题必须包含关键词
主题 intitle:关键字

allintitle:
搜索返回的是页面标题中包含多组关键词的文件。

url

inurl:
连接中必须包含关键词

inurl：任意字符串
allinurl

视频

视频名称 thunder
视频名称 torrent（种子）、Magnet（磁力）、download（下载）

音乐

关键字 inurl:mp3

图片

百度识图、谷歌识图/图搜
XXX inurl:photo

s +鼠标右键 ——图片详细信息

site:

关键词 site:baidu.com
禁止 http://；“www”前缀

filetype:

指定特定内容格式
关键词 filetype:格式
doc、xls、ppt、pdf、rtf、all。。。

related:

http://xxxx.com
返回的结果是与某个网站有关联的页面

精准匹配 “”

内容筛选

*
通配搜索

OR

A -B
包括A,不包括B

A +B
包括A、B

A|B
不太精准 A或B

数字a..数字b
数字范围
​上限a—-下限b

空格
既…，又…

组合

site:http://xxx.com filetype:pdf
filetype :csv site:某网盘 或资源站点 inurl:某url特征。

…

垂直网站搜索（专业平台）

git

感

版本控制：记录每次文件的改动，任意滚动时间条。
分支管理，多人协同。建立无数个分支，在分支上进行修改迭代，合并分支，再推送至

win环境配置

编辑文件不要用记事本

安装git

msysgit

打开 git bash 输入

git config –global user.name “apaat-kcalm”
git config –global user.email “401671898@qq.com”

常用命令

mkdir+名
新建目录

cd+名
转到目录

pwd
显示当前目录

git status
查看当前仓库状态

git diff
查看做了什么不同的修改

git log
查看所有历史
–pretty=oneline
简要信息

ls 查看已有文件

cat 文件名
查看文件全部内容

本地

创建本地仓库（repository）

初始化

git init
当前目录多出.git文件夹

编辑文件

工作区==电脑文件夹

git add 文件名+后缀

工作区的文件添加到暂存区

Changes to be committed:

git commit -m”注释”

从暂存区提交到版本库master
nothing to commit (working directory clean)
可一次提交多个文件

版本

回退

git reset –hard HEAD^

  • HEAD——版本
  • ^上一个
  • ^^上上个

版本号回退

  • git reflog 查看所有版本号

  • git reset –hard 版本号

    撤销修改（提交之前）

    git checkout – file

  • 修改，但未add
    回到和版本库（提交）状态

  • 已经add，再做修改
    回到add状态

    删除

    文件目录删除
    git rm file

    分支

    合并到主分支master再推送

    命令

    git branch
    查看当前分支

git checkout -b 分支名
创建并切换

git branch 分支名
创建分支

git checkout 分支名
切换分支

git merge
分支合并到master

git merge –no-ff -m “注释” dev
合并后的历史有分支

git branch -d 分支名
删除分支

远程仓库

（已建好本地pro）关联远程仓库

本地与远程同步~

（SSH）配置

ssh-keygen -t rsa -C “401671898@qq.com”
一路回车

（github页面）添加SSH
id_rsa.pub

验证
:ssh -T git@github.com
yes

github新建仓库

git bash

git remote add origin git@github.com:apaat-kcalm/仓库名.git
关联一个远程库

git push -u origin master
第一次推送master分支的所有内容
-u 关联本地和远程master

git push origin 分支名

推送新的修改

创建远程仓库

/git remote
查看远程库的信息

git remote –v
详细信息

克隆
git clone git@github.com:apaat-kcalm/仓库名.git

git push origin 分支名
推送

分支冲突—推送失败

  • git pull
    ​抓取最新提交

git pull提示“no tracking information”
本地分支和远程分支的链接关系没有创建
用命令git branch –set-upstream branch-name origin/branch-name。
